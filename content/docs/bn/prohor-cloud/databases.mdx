প্রহরী ক্লাউড দিচ্ছে সম্পূর্ণ পরিচালিত **DBaaS (Database-as-a-Service)** সলিউশন। আমরা আপনার প্রজেক্টের ডাটাবেসের দৈনিক ব্যাকআপ, রেপ্লিকেশন এবং কানেকশন পুলিং হ্যান্ডেল করি।

## ডাটাবেস প্রোভিশন করা

প্রহরী CLI ব্যবহার করে কাস্টম পোস্টগ্রেস ডাটাবেস তৈরি করুন:

```bash
prohor db create postgres startup-db
```

ডাটাবেস তৈরি হয়ে যাওয়ার পর কানেকশন প্যারামিটারসমূহ টার্মিনালে দেখতে পাবেন।

## নিরাপত্তা ও ব্যাকআপ

- **এনক্রিপ্টেড সংযোগ**: সমস্ত কানেকশন ডাটাবেস ট্রাফিকের জন্য SSL এনক্রিপশন আবশ্যক।
- **দৈনিক ব্যাকআপ**: ডাটাবেস স্ন্যাপশটগুলো সম্পূর্ণ আলাদা সুরক্ষিত ডেটা সেন্টারে সংরক্ষণ করা থাকে।
