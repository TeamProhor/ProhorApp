আপনার অ্যাপ্লিকেশনের সিক্রেট এপিআই চাবি এবং কনফিগারেশন ভেরিয়েবলসমূহ প্রহরীর সিকিউর ক্লাউড ভল্টে নিরাপদে সংরক্ষণ করুন।

## ভেরিয়েবল সেট করা

CLI এর মাধ্যমে খুব সহজে যেকোনো এনভায়রনমেন্ট সিক্রেট সেট করতে পারেন:

```bash
prohor env set API_SECRET_KEY=12345
```

এই সিক্রেট ডাটা সম্পূর্ণ এনক্রিপ্ট হয়ে আপনার চলমান অ্যাপ্লিকেশনের কন্টেইনারে ইনজেক্ট হবে।

## ডিফল্ট ভেরিয়েবলসমূহ

আমাদের প্রহরী ক্লাউড সিস্টেম স্বয়ংক্রিয়ভাবে নিম্নলিখিত ভেরিয়েবলগুলো আপনার অ্যাপ্লিকেশনে ইনজেক্ট করে:
- `PORT`: কন্টেইনার যে পোর্টে রান করবে (ডিফল্ট: `3000`)।
- `NODE_ENV`: প্রোডাকশন সার্ভারে এটি `production` হিসেবে সেট থাকে।
