প্রহরী কুইকস্টার্ট গাইডে আপনাকে স্বাগতম। আপনার আইডিয়াকে বাস্তবে রূপ দিতে এবং আমাদের প্ল্যাটফর্ম কনসোল ব্যবহার করতে এই ধাপগুলো অনুসরণ করুন।

## ধাপ ১: আইডিয়া পিচ করুন

আপনার বিজনেস আইডিয়া বা ভিশন আমাদের টিমকে জানাতে CLI ব্যবহার করতে পারেন:

```bash
curl -fsSL https://api.prohor.com/pitch.sh | bash -s -- --idea "your-next-unicorn"
```

অথবা ওয়েব কনসোলের "Book Call" বাটনে ক্লিক করে সরাসরি মিটিং শিডিউল করুন।

## ধাপ ২: প্রহরী কনসোলে লগ ইন করুন

আপনার অ্যাপ ডেভেলপমেন্ট ট্র্যাক, ডেটাবেস প্রোভিশন বা মার্কেটিং ক্যালেন্ডার দেখতে লগ ইন করুন:

```bash
prohor auth login
```

## ধাপ ৩: এমভিপি (MVP) বিল্ড ট্র্যাক করুন

আপনার অ্যাপের ক্লাউড ডেপ্লয়মেন্ট এবং বর্তমান কোডিং স্ট্যাটাস ট্র্যাক করতে:

```bash
prohor status --project my-startup-mvp
```

কনসোলে ঢুকলে আপনি স্বয়ংক্রিয় সিকিউরিটি স্ক্যান রিপোর্ট এবং এপিআই কানেকশন দেখতে পাবেন!
