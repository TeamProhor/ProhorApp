ProhorCloud is an automated PaaS (Platform as a Service) and DBaaS (Database as a Service) solution. It reads your GitHub codebase, builds Docker containers, provisions databases, and automatically configures SSL and routing. Available via Web Dashboard and CLI.

ProhorCloud প্ল্যাটফর্মটি স্বয়ংক্রিয়ভাবে আপনার অ্যাপ্লিকেশন হোস্ট করতে সাহায্য করে। এটি আপনার প্রজেক্ট বুঝে নিয়ে কন্টেইনারাইজ করে এবং সিকিউরভাবে ডেপ্লয় করে।

## Get started

Choose your environment to get started. Most surfaces require a ProhorCloud account. The CLI also supports connecting to your own self-hosted infrastructure.
