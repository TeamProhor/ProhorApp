Welcome to the Quickstart guide. Follow these simple steps to deploy your first application to ProhorCloud.

## Step 1: Install the CLI

You can install the CLI using our installation script:

```bash
curl -fsSL https://cli.prohorcloud.com/install.sh | bash
```

## Step 2: Login to your account

Authenticate your CLI with your ProhorCloud account:

```bash
prohor login
```

## Step 3: Deploy your project

Initialize and deploy a new project:

```bash
prohor deploy
```
