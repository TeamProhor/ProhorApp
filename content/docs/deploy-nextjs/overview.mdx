Deploying a Next.js application to ProhorCloud is simple and fully optimized. ProhorCloud automatically detects Next.js projects and configures the optimal build and caching settings.

## Automated Detection

When you connect your GitHub repository, ProhorCloud parses your `package.json` to identify dependencies.

```json
{
  "dependencies": {
    "next": "latest",
    "react": "latest"
  }
}
```

## Production Optimization

All Next.js deployments benefit from:
- Edge routing and caching for static files.
- Automated image optimization support.
- Zero-downtime rolling updates.
