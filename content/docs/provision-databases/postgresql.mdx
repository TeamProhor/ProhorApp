ProhorCloud provides fully managed PostgreSQL instances with automated backups, connection pooling, and resource scaling.

## Provisioning

To provision a new database via CLI:

```bash
prohor db create postgres my-db
```

Or use the Web Dashboard and click "Create Database".

## Connection URL

Once provisioned, your application will receive a secure connection string:

```
postgresql://user:password@host:port/database?sslmode=require
```

This URL is automatically injected into your app's environment variables as `DATABASE_URL`.
