**Prohor Cloud** provides automated web and application hosting with instant zero-downtime updates, global content delivery networks, and automatic SSL setup.

## Hosting Features

Every web app deployed on Prohor Cloud receives premium infrastructure resources:

- **Git Integration**: Push your code to GitHub, and our build pipelines build and deploy it instantly.
- **Edge Routing**: High-speed edge networks cache assets closer to your users.
- **Automatic SSL**: Safe HTTPS encryption configured instantly for custom domains.

## Deploying an App

Deploy your web application directly from your terminal:

```bash
prohor deploy --dir ./my-app
```
