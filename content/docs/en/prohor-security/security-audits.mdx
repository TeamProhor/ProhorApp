---

## Security Auditing Services

We conduct regular penetration testing, server scans, and code verification for all client assets:
- **Penetration Testing**: Simulating real-world cyber attacks to find vulnerabilities.
- **Code Audits**: Static and dynamic codebase analysis to check for security flaws.
- **Data Protection**: Strict database firewalls, access controls, and TLS/SSL configurations.

## Running Automated Vulnerability Scans

Initiate an automated security scan directly from the CLI:
```bash
prohor security scan --project my-mvp
```
